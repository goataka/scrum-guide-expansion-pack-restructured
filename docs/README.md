## 📖 オリジナルからの変更点

この再構成版では、オリジナルのコンテンツを以下のように**書式化・整理**しました：

### 適用した変更 ✅

- **箇条書きと太字の強調**: 重要なポイントを箇条書きで整理し、キーワードを太字で強調
- **セクションごとの要約**: 各セクションの冒頭に `> [!NOTE]` 形式で要約を追加
- **サブセクションの追加**: 長いセクションを読みやすいh3サブセクションに分割
- **ナビゲーションリンク**: 各ページに「← 前へ | 目次 | 次へ →」リンクを追加
- **16ファイル構成**: トピックごとに独立したファイルに分割し、特定の情報を素早く参照可能に

### 変更していないこと ❌

- **コンテンツの意味**: 原文の内容は一切変更なし
- **情報の削除**: すべてのオリジナル情報を保持
- **翻訳**: 日本語訳はそのまま使用



**オリジナルスクラムガイド**: Ken Schwaber & Jeff Sutherland
