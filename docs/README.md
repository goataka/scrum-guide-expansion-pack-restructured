# 構造化について

このリポジトリは、[Scrum Guide Expansion Pack](https://scrumexpansion.org/ja/scrum-guide-expansion-pack/2025.6/)の日本語版を、読みやすさと参照のしやすさを向上させるために再構成したものです。

## 実施した変更

### 1. ファイルの分割

元の単一ファイル（4,135行）を、論理的な9つのセクションに分割しました：

1. **01-background.md** (129行) - 背景と目的
2. **02-scrum-basics.md** (715行) - スクラムの基本概念
3. **03-theory.md** (316行) - 支援・補完理論の詳細
4. **04-roles.md** (545行) - スクラムの役割
5. **05-artifacts.md** (294行) - スクラムの作成物
6. **06-events.md** (255行) - スクラムイベント
7. **07-multi-team.md** (96行) - 複数チームでのスクラム
8. **08-conclusion.md** (166行) - まとめと要約
9. **09-appendix.md** (1,701行) - 付録、用語集、参考文献

### 2. ナビゲーションの追加

各ドキュメントファイルに以下を追加しました：

- **ヘッダーナビゲーション**: 前のページ、目次、次のページへのリンク
- **フッターナビゲーション**: ページ末尾でも同様のナビゲーション
- これにより、順次読み進めることも、目次から必要なセクションに直接アクセスすることも可能

### 3. ライセンス情報の明確化

- **NOTICE.md**: 詳細なライセンスと帰属情報を記載
- **各ファイル**: CC BY-SA 4.0ライセンス通知を追加
- **README.md**: 簡潔なライセンス情報と、詳細情報へのリンクを記載

### 4. 目次の作成

README.mdに、各セクションの主要トピックを含む包括的な目次を作成しました。

## 内容の変更について

### 変更していないこと

- **コンテンツの意味や内容**: 元のテキストは一切変更していません
- **翻訳**: 日本語訳はそのまま保持しています
- **オリジナルの著作権**: すべて元の著者と翻訳者に帰属しています

### 変更したこと

- **構造**: 単一ファイルから複数ファイルへの分割（構造的な変更のみ）
- **ナビゲーション**: 読みやすさを向上させるためのナビゲーション要素の追加
- **ライセンス表示**: より明確で目立つライセンス情報の配置
- **箇条書き化**: 長い段落を読みやすい箇条書き形式に変換
- **重要箇所の強調**: 重要な概念やキーワードを太字で強調
- **要約の追加**: 各主要ブロックの直下に約300字の要約を追加
- **階層化**: 内容を分類可能な場合、より詳細な階層（サブセクション）を追加
  - 📝 マークは、読みやすさ向上のために追加されたサブセクションを示します

## なぜこの構造化を行ったか

1. **可読性の向上**: 4,000行以上の単一ファイルより、トピックごとに分かれたファイルの方が読みやすい
2. **参照の容易さ**: 特定のトピック（例：役割や作成物）を素早く見つけられる
3. **学習体験の向上**: 順次読み進めることも、必要な部分だけを参照することも可能
4. **ライセンス遵守**: CC BY-SA 4.0の要件を満たすため、明確な帰属情報を提供

## ライセンスコンプライアンス

この再構成は、CC BY-SA 4.0ライセンスの要件を完全に満たしています：

- ✅ **帰属 (Attribution)**: すべてのファイルに著者と翻訳者の情報を記載
- ✅ **継承 (ShareAlike)**: コンテンツは元と同じCC BY-SA 4.0ライセンスで提供
- ✅ **変更通知**: NOTICE.mdとREADME.mdに構造的変更について明記
- ✅ **元の著作権**: オリジナルの著作権情報を保持

## 使用方法

### 読む順序

1. 初めて読む場合は、[背景と目的](docs/01-background.md)から順に読むことを推奨
2. 特定のトピックを参照したい場合は、[README.md](README.md)の目次から直接アクセス
3. 各ページのナビゲーションを使って、前後のページに移動可能

### 再利用

このコンテンツを再利用する場合：

1. CC BY-SA 4.0の条件に従ってください（詳細は[NOTICE.md](NOTICE.md)参照）
2. オリジナルの著者と翻訳者を明記してください
3. 変更を加えた場合は、その旨を明記してください
4. 同じCC BY-SA 4.0ライセンスで公開してください
